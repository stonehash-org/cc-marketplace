# cc-mkp

개인용 Claude Code 플러그인 마켓플레이스입니다.
프로젝트 관리, 생산성 도구 등 자주 사용하는 플러그인을 모아 관리합니다.

---

## 설치 방법

Claude Code에서 아래 명령어를 입력하면 마켓플레이스가 등록됩니다.

```bash
/plugin marketplace add jaybee-sths/cc-mkp
```

등록 후 개별 플러그인을 설치할 수 있습니다.

---

## 플러그인 목록

| 플러그인 | 설명 | 버전 |
|----------|------|------|
| **clickup** | 자연어로 ClickUp 워크스페이스를 관리 (태스크, 스프린트, 문서, 타임트래킹 등) | 1.0.0 |

## ClickUp 플러그인

### 개요

ClickUp의 공식 MCP 서버를 통해 Claude Code에서 자연어로 ClickUp을 조작할 수 있습니다.
태스크 생성/수정/검색, 워크스페이스 탐색, 타임트래킹 등 50개 이상의 기능을 지원합니다.

### 설치

```bash
/plugin install clickup@cc-mkp
```

### 인증 방식: OAuth (자동)

공식 ClickUp MCP 서버는 **OAuth 인증**을 사용합니다.
별도의 API Key 설정이 필요 없으며, 최초 연결 시 브라우저에서 ClickUp 로그인 화면이 나타납니다.

> 이전 버전에서 사용하던 `CLICKUP_API_KEY` 환경변수는 더 이상 필요하지 않습니다.

### 사용 예시

Claude Code에서 자연어로 요청하면 됩니다.

#### 워크스페이스 탐색

```
"워크스페이스 구조 보여줘"
"Project Management 스페이스의 리스트 목록 알려줘"
"멤버 목록 조회해줘"
```

#### 태스크 관리

```
"Between Wallet 리스트에 새 태스크 만들어줘"
"태스크 #abc123 상세 내용 보여줘"
"이번 스프린트 태스크 목록 보여줘"
"태스크 상태를 완료로 변경해줘"
"태스크에 코멘트 추가해줘"
```

#### 검색

```
"결제 관련 태스크 검색해줘"
"나한테 할당된 태스크 보여줘"
"이번 주 마감인 태스크 찾아줘"
```

#### 타임트래킹

```
"타이머 시작해줘"
"현재 진행 중인 타이머 확인해줘"
"타이머 멈춰줘"
```

#### 문서

```
"새 문서 만들어줘"
"문서 목록 보여줘"
"문서에 페이지 추가해줘"
```

### 지원 기능 요약

| 카테고리 | 주요 기능 |
|----------|----------|
| **워크스페이스** | 구조 조회, 멤버 목록, 멤버 검색 |
| **태스크** | 생성, 조회, 수정, 이동, 복제, 삭제, 벌크 작업 |
| **검색** | 키워드 검색, 담당자/생성자/상태/날짜 필터링 |
| **리스트/폴더** | 생성, 조회, 수정, 삭제 |
| **태그** | 생성, 수정, 삭제, 태스크에 태그 추가/제거 |
| **타임트래킹** | 타이머 시작/중지, 수동 입력, 조회 |
| **문서** | 문서 생성, 페이지 추가/수정, 목록 조회 |
| **채팅** | 채널 생성, 메시지 전송/조회 |
| **코멘트** | 태스크 코멘트 추가/조회 |
| **파일** | 태스크에 파일 첨부 |

### ClickUp 계층 구조 참고

```
Workspace (팀)
  └─ Space (프로젝트 영역)
       └─ Folder (선택적 그룹)
            └─ List (태스크 컨테이너)
                 └─ Task (작업 항목)
                      ├─ Subtask (하위 작업)
                      └─ Checklist (체크리스트)
```

### 주의 사항

- **상태 변경 전** 해당 리스트의 사용 가능한 상태 목록을 먼저 확인합니다 (리스트마다 커스텀 상태가 다름)
- **태스크 삭제는 영구적**입니다. 가능하면 닫기/보관을 권장합니다
- **벌크 수정** 시 각 태스크 업데이트마다 워쳐에게 알림이 발송될 수 있습니다
- **설명(Description) 수정** 시 원본을 먼저 표시합니다 (ClickUp에는 되돌리기 기능이 없음)

## 새 플러그인 추가 방법

이 마켓플레이스에 새로운 플러그인을 추가하려면:

1. `plugins/플러그인명/` 디렉토리 생성
2. `.claude-plugin/plugin.json` 매니페스트 파일 작성
3. 필요에 따라 skills, commands, agents, hooks 추가
4. `.claude-plugin/marketplace.json`에 플러그인 등록

### 디렉토리 구조 예시

```
cc-mkp/
├── .claude-plugin/
│   └── marketplace.json        # 마켓플레이스 매니페스트
├── plugins/
│   └── clickup/                # 플러그인 디렉토리
│       ├── .claude-plugin/
│       │   └── plugin.json     # 플러그인 매니페스트
│       ├── .mcp.json           # MCP 서버 설정
│       └── skills/             # 스킬 정의
│           └── clickup/
│               ├── SKILL.md
│               └── references/
└── README.md
```

## 문제 해결

### ClickUp 연결이 안 될 때

1. Claude Code 재시작 후 `claude mcp list`로 연결 상태 확인
2. 연결 실패 시 캐시 초기화: `~/.claude/plugins/cache/` 하위의 clickup 캐시 삭제 후 재시작
3. OAuth 인증 만료 시 브라우저에서 다시 로그인

### 플러그인이 보이지 않을 때

```bash
# 마켓플레이스 재등록
/plugin marketplace remove cc-mkp
/plugin marketplace add jaybee-sths/cc-mkp
```
