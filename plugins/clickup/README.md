# ClickUp 플러그인

자연어로 ClickUp 워크스페이스를 관리하는 Claude Code 플러그인입니다.
ClickUp 공식 MCP 서버를 통해 태스크, 문서, 타임트래킹 등 50개 이상의 기능을 지원합니다.

---

## 설치

```bash
/plugin install clickup@cc-mkp
```

---

## 인증

공식 ClickUp MCP 서버는 **OAuth 인증**을 사용합니다.

- 별도의 API Key 설정이 필요 없습니다
- 최초 연결 시 브라우저에서 ClickUp 로그인 화면이 나타납니다
- 로그인하면 자동으로 연결됩니다

> 이전 버전에서 사용하던 `CLICKUP_API_KEY` 환경변수는 더 이상 필요하지 않습니다.

---

## 사용 예시

Claude Code에서 자연어로 요청하면 됩니다.

### 워크스페이스 탐색

```
"워크스페이스 구조 보여줘"
"Project Management 스페이스의 리스트 목록 알려줘"
"멤버 목록 조회해줘"
```

### 태스크 관리

```
"Between Wallet 리스트에 새 태스크 만들어줘"
"태스크 #abc123 상세 내용 보여줘"
"이번 스프린트 태스크 목록 보여줘"
"태스크 상태를 완료로 변경해줘"
"태스크에 코멘트 추가해줘"
```

### 검색

```
"결제 관련 태스크 검색해줘"
"나한테 할당된 태스크 보여줘"
"이번 주 마감인 태스크 찾아줘"
```

### 타임트래킹

```
"타이머 시작해줘"
"현재 진행 중인 타이머 확인해줘"
"타이머 멈춰줘"
```

### 문서

```
"새 문서 만들어줘"
"문서 목록 보여줘"
"문서에 페이지 추가해줘"
```

## 지원 기능

| 카테고리 | 주요 기능 |
|----------|----------|
| **워크스페이스** | 구조 조회, 멤버 목록, 멤버 검색 |
| **태스크** | 생성, 조회, 수정, 이동, 복제, 삭제, 벌크 작업 |
| **검색** | 키워드 검색, 담당자/생성자/상태/날짜 필터링 |
| **리스트/폴더** | 생성, 조회, 수정, 삭제 |
| **태그** | 생성, 수정, 삭제, 태스크에 태그 추가/제거 |
| **타임트래킹** | 타이머 시작/중지, 수동 입력, 조회 |
| **문서** | 문서 생성, 페이지 추가/수정, 목록 조회 |
| **채팅** | 채널 생성, 메시지 전송/조회 |
| **코멘트** | 태스크 코멘트 추가/조회 |
| **파일** | 태스크에 파일 첨부 |

## ClickUp 계층 구조

```
Workspace (팀)
  └─ Space (프로젝트 영역)
       └─ Folder (선택적 그룹)
            └─ List (태스크 컨테이너)
                 └─ Task (작업 항목)
                      ├─ Subtask (하위 작업)
                      └─ Checklist (체크리스트)
```

---

## 주의 사항

- **상태 변경 전** 해당 리스트의 사용 가능한 상태 목록을 먼저 확인합니다 (리스트마다 커스텀 상태가 다름)
- **태스크 삭제는 영구적**입니다. 가능하면 닫기/보관을 권장합니다
- **벌크 수정** 시 각 태스크 업데이트마다 워쳐에게 알림이 발송될 수 있습니다
- **설명(Description) 수정** 시 원본을 먼저 표시합니다 (ClickUp에는 되돌리기 기능이 없음)

## 문제 해결

### 연결이 안 될 때

1. Claude Code 재시작 후 연결 상태 확인
   ```bash
   claude mcp list
   ```
2. 연결 실패 시 캐시 초기화: `~/.claude/plugins/cache/` 하위의 clickup 캐시 삭제 후 재시작
3. OAuth 인증 만료 시 브라우저에서 다시 로그인

### MCP 설정 확인

현재 사용 중인 MCP 설정 (`.mcp.json`):

```json
{
  "mcpServers": {
    "clickup": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://mcp.clickup.com/mcp"]
    }
  }
}
```
